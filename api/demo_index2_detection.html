Duplicate Detection in index2.php

This shows exactly how the duplicate detection works when you create a citation.

YES! It WILL Detect Duplicates in index2.php

Status: ✓ ACTIVE & WORKING

Script Loaded: /tmg/assets/js/duplicate-detection.js

API Endpoint: /tmg/api/check_duplicates.php

How It Works - Step by Step

Step 1: You Open the Citation Form

Go to: http://localhost/tmg/public/index2.php

The duplicate-detection.js script loads automatically

Event listeners are attached to form fields

Step 2: You Start Typing Driver Information

When you type in ANY of these fields and click away (blur):

  • License Number (license_number)
  • Plate Number (plate_mv_engine_chassis_no)
  • First Name (first_name)
  • Last Name (last_name)
  • Date of Birth (date_of_birth)
  • Barangay (barangay)

The system waits 500ms (half a second) after you stop typing

Step 3: Automatic Duplicate Check

The system sends an AJAX request to /tmg/api/check_duplicates.php

POST /tmg/api/check_duplicates.php Content-Type: application/json { "license_number": "", "plate_number": "", "first_name": "RICHMOND", "last_name": "ROSETE", "date_of_birth": "1999-10-17", "barangay": "" }
Step 4: System Searches for Duplicates

The system checks using 3 strategies:

  1. License Number Match (100% confidence) - Exact license match
  2. Plate Number Match (85% confidence) - Same vehicle
  3. Fuzzy Name Matching (30-80% confidence) - Similar names + DOB + Barangay
Step 5a: If Duplicates Found

A WARNING MODAL APPEARS!

⚠ Possible Duplicate Driver Found

We found existing records that may match this driver.

The modal shows:

  • All matching drivers with confidence scores
  • Their citation history
  • Total offenses
  • Two options:
    • Use existing driver record (if same person)
    • Create new record (if different person)
Step 5b: If No Duplicates Found

Nothing happens - you can continue filling the form normally

Step 6: You Select Violations

When you check violation checkboxes:

  • System automatically looks up offense history for this driver
  • Auto-selects correct offense count (1st, 2nd, or 3rd offense)
  • Shows notification if 2nd or 3rd offense detected
